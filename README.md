Library Management System in Java

A console-based Library Management System built using Java and Object-Oriented Programming principles.

Features:-

Add new books to the library, view all available books, search books by title, search books by author, delete books from the library, reduce book quantity, user management with auto-generated IDs, admin authentication for secure operations, issue books only through admin login, prevent duplicate book issuance, check book availability before issuing, persistent data storage using serialization, and automatic data recovery using deserialization.

Technologies Used:-

Java, Object-Oriented Programming (OOP), ArrayList, File Handling, Serialization & Deserialization, and Exception Handling.

Project Structure:-

Main.java – Main application logic, FileUtil.java – Handles serialization and deserialization, bookDetails.txt – Stores book data, userDetails.txt – Stores user data, and issueBook.txt – Stores issued book records.

Future Enhancements:-

Return book functionality, fine calculation system, file encryption for enhanced security, graphical user interface using JavaFX or Swing, and a web-based version using Spring Boot.
